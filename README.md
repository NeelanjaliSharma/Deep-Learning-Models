# 🧠 Deep Learning Models

This repository contains my implementations of fundamental Deep Learning models using Python. It focuses on building a strong understanding of neural networks through practical notebooks covering Artificial Neural Networks (ANN), Convolutional Neural Networks (CNN), Recurrent Neural Networks (RNN), and Long Short-Term Memory (LSTM). Each file demonstrates the complete workflow including data preprocessing, model building, training, and evaluation. This repository represents my learning journey in Deep Learning and helps strengthen both theoretical concepts and hands-on skills.

## 🛠 Tech Stack

Python, NumPy, Pandas, Matplotlib, TensorFlow/Keras (or PyTorch), Jupyter Notebook

## 🚀 Goal

To build a strong foundation in Deep Learning and apply it to real-world problems.
